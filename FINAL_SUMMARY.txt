HOW TO GET STARTED - 3 SIMPLE STEPS
================================================================================

STEP 1: Get Groq API Key (2 minutes)
────────────────────────────────────
1. Visit: https://console.groq.com
2. Sign up (email or GitHub, no credit card)
3. Go to API Keys section
4. Create new API key
5. Copy the key (starts with gsk_)

STEP 2: Setup Project (5 minutes)
─────────────────────────────────
1. Clone repository
2. Copy .env.example to .env
3. Add GROQ_API_KEY to .env
4. Run: pip install -r requirements.txt
5. Done!

STEP 3: Run System (2 minutes)
──────────────────────────────
Terminal 1:
cd rag_system/backend && python main.py

Terminal 2:
cd rag_system/frontend && streamlit run app.py

TOTAL TIME: ~10 minutes from zero to running RAG system
