QUICK SETUP - 10 MINUTES TO RUNNING RAG SYSTEM
================================================================================

STEP 1: Get Groq API Key (2 minutes)
────────────────────────────────────
1. Go to: https://console.groq.com
2. Click "Sign Up" (email or GitHub)
3. No credit card required
4. Navigate to "API Keys" section
5. Click "Create API Key"
6. Copy the key (format: gsk_xxxxx...)

This is your gateway to fast, free LLM inference. 
Groq boasts the world's fastest LLM API.

STEP 2: Clone & Configure (3 minutes)
──────────────────────────────────────
git clone <repository>
cd enterprise-knowledge-assistant

# Create environment file
cp .env.example .env
nano .env

# Edit .env and add your key:
GROQ_API_KEY=<your_key_here>

Save and exit.

STEP 3: Install Dependencies (3 minutes)
─────────────────────────────────────────
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt

Wait for installation to complete.

STEP 4: Start the System (2 minutes)
────────────────────────────────────
Terminal 1 - Backend:
cd rag_system/backend
python main.py

You should see: "Application startup complete"

Terminal 2 - Frontend:
cd rag_system/frontend
streamlit run app.py

Streamlit opens browser automatically.

STEP 5: Test It Out (1 minute)
───────────────────────────────
1. In Streamlit UI, click "Upload" tab
2. Upload a PDF, DOCX, or TXT file
3. Click "Q&A" tab
4. Ask a question about your document
5. Get answer with sources!

DONE! Your RAG system is running.

Automatic Fallback Chain:

If primary model is slow, system tries faster alternatives automatically.

================================================================================
TROUBLESHOOTING
================================================================================

Issue: "GROQ_API_KEY not set"
───────────────────────────────
Solution:
1. Set environment variable:
   export GROQ_API_KEY="your_key"
2. Or add to .env file:
   GROQ_API_KEY=your_key
3. Verify it's set:
   echo $GROQ_API_KEY

Issue: "Invalid API Key"
─────────────────────────
Solution:
1. Go to console.groq.com
2. Check API Keys section
3. Verify key is correct
4. Try regenerating the key
5. Copy without whitespace

Issue: "All fallback models failed"
────────────────────────────────────
Solution:
1. Check internet connection
2. Visit status.groq.com
3. Wait a few minutes and retry
4. Check API key validity
5. Contact Groq support

Issue: "Slow responses"
────────────────────────
Solution:
1. Typical: 10-50 seconds (normal)
2. Check internet speed
3. Check Groq status
4. Reduce max_tokens
5. Try fallback model (smaller, faster)

Issue: "Rate limit exceeded"
──────────────────────────────
Solution:
1. Wait a few minutes
2. Upgrade to Pro tier
3. Use multiple API keys
4. Implement throttling
